# SimView - Non-default constructions
Not all constructions can be attached as default constructions in the various construction groups, of course.

A construction is attached or modified in the same way as default constructions are attached. The database is opened and the desired construction (SfB number) is dragged from the database to the construction in the model's tree summary to which it is to be attached.
