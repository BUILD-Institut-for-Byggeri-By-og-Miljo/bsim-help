# Shortcut keys

| Shortcut        | Function                                                                 | Program               |
|-----------------|--------------------------------------------------------------------------|-----------------------|
| gray +          | Zoom in on graphical view of the model                                   | SimView, XSun         |
| gray -          | Zoom out from graphical view of the model                                | SimView, XSun         |
| Ctrl-A          | Auto-scale graphical view of the model                                   | SimView               |
| Ctrl-C          | Copy                                                                     |                |
| Ctrl-N          | Start wizard to create a new model                                       | SimView               |
| Ctrl-O          | Open an existing project                                                 | SimView               |
| Ctrl-P          | Print out documentation for the model                                    | SimView (ModelList)   |
| Ctrl-S          | Save the model                                                           | SimView               |
| Ctrl-V          | Paste copy                                                               |                |
| DELETE          | Delete current object                                                    | SimView               |
| Down arrow      | Move eye in graphical view up                                            | SimView, XSun         |
| F1              | Context sensitive help to the active window. To make sure this window is the active, left-click the desired window. | BSim              |
| Shift-F1        | Context-sensitive help (not currently working)                           |                       |
| F6              | Go to next window                                                        | SimView               |
| Shift-F6        | Return to previous window                                                | SimView               |
| Ctrl-HOME       | Display next period                                                      | tsbi5                 |
| Ctrl-INSERT     | Copy                                                                     |                       |
| Shift-INSERT    | Paste copy                                                               |                       |
| Left arrow      | Move eye in graphical view to the right                                  | SimView, XSun         |
| Ctrl-left arrow | Display previous period                                                  | tsbi5                 |
| Right arrow     | Move eye in graphical view to the left                                   | SimView, XSun         |
| Up arrow        | Move eye in graphical view down                                          | SimView, XSun         |
| Shift-down arrow| Moves to the next / previous object in the tree                          | SimView, tsbi5, XSun  |
| Shift-up arrow  | Moves to the next / previous object in the tree                          | SimView, tsbi5, XSun  |
| Ctrl-X          | Delete current object                                                    | SimView               |
| Ctrl-Z          | Undo previous change of geometry                                         | SimView               |
| Alt-ENTER       | Opens the property dialog for the highlighted object in the tree, when the tree is active | SimView, tsbi5, XSun |
| Alt-b           | Insert building                                                          | SimView               |
| Alt-c           | Remove highlighting of selected objects                                  | SimView               |
| Alt-e           | Create an edge from two selected corners (vertex)                        | SimView               |
| Alt-f           | Add face from three corners or face and corner in same space             | SimView               |
| Alt-i           | Insert (fill in) a WinDoor or an opening in the middle of a face          | SimView               |
| Alt-m           | Move the current face                                                    | SimView               |
| Alt-n           | Jump to next time step                                                   | XSun                  |
| Alt-o           | Insert opening in current face from a datum point and local x-axis       | SimView               |
| Alt-p           | Jump to previous time step                                               | XSun                  |
| Alt-r           | Add a space to outside of current face                                   | SimView               |
| Alt-w           | Insert WinDoor in current face from a datum point and local x-axis       | SimView               |
| Alt-x           | Exports the actual results table from tsbi5. Tables as a tab separated text-file, e.g. for import in a spread-sheet program. The table is saved as hourly values over the selected period. | tsbi5 |
