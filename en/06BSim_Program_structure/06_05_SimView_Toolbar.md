# SimView - Toolbar

<figure id="image_linewidth">
<img src="./assets/ToolBar.GIF" alt="Toolbar in BSim.">
<figcaption>Toolbar in BSim.</figcaption>
</figure>

The toolbar in *BSim* contains a number of buttons with the following functions (from left to right):

<img src="./assets/1SimView - Værktøjsbjælken.png" width=15> Create new project (shortcut: Ctrl + N).
<br>
<img src="./assets/2SimView - Værktøjsbjælken.png" width=15> Open existing project (shortcut: *Ctrl + O*). In the open dialog box models (from version 1.1.12.12) are shown with a coloured icon <img src="./assets/OPEN_DIS.GIF" width=15> and backup-models with a grey icon <img src="./assets/grå hus.gif" width=15>.
<br>
<img src="./assets/3SimView - Værktøjsbjælken.png" width=15> Save current project (shortcut: *Ctrl + S*).
<br>
<img src="./assets/4SimView - Værktøjsbjælken.png" width=15> Expand tree summary from current level down.
<br>
<img src="./assets/5SimView - Værktøjsbjælken.png" width=15> Collapse tree summary from current level down.
<br>
<img src="./assets/6SimView - Værktøjsbjælken.png" width=15> Undo last action (shortcut: *Ctrl + Z*).
<br>
<img src="./assets/7SimView - Værktøjsbjælken.png" width=15> Redo the last undo
<br>
<img src="./assets/8SimView - Værktøjsbjælken.png" width=15> Cut (shortcut: *Ctrl + X*).
<br>
<img src="./assets/9SimView - Værktøjsbjælken.png" width=15> Copy (shortcut: *Ctrl + C*).
<br>
<img src="./assets/10SimView - Værktøjsbjælken.png" width=15> Paste (shortcut: *Ctrl + V*).
<br>
<img src="./assets/11SimView - Værktøjsbjælken.png" width=15> Print preview.
<br>
<img src="./assets/12SimView - Værktøjsbjælken.png" width=15> Print (shortcut: *Ctrl + P*).
<br>
<img src="./assets/13SimView - Værktøjsbjælken.png" width=15> Show the model using the standard internet browser in [model DOC](../24Miscellaneous/24_29_Model_DOC.md) format.
<br>
<img src="./assets/14SimView - Værktøjsbjælken.png" width=15> Show listing (documentation) for current view, [*ModelList*](../06BSim_Program_structure/06_07_SimView_Printing_a_model.md).
<br>
<img src="./assets/15SimView - Værktøjsbjælken.png" width=15> Converts the building model to be used in the Danish building code energy frame compliance checker, Be18 (available in English and using customized weather data). 
<br>
<img src="./assets/16SimView - Værktøjsbjælken.png" width=15> Open BEAT to perform environmental analyses.
<br>
<img src="./assets/17SimView - Værktøjsbjælken.png" width=15> Calculation of daylight with [*SimLight*](../15SimLight_Daylight_calculations/15_01_Daylight_calculations_with_SimLight.md).
<br>
<img src="./assets/18SimView - Værktøjsbjælken.png" width=15> Switch to [Switch to *SimView*.](../09SimView/09_01_SimView.md).
<br>
<img src="./assets/19SimView - Værktøjsbjælken.png" width=15> Switch to [*XSun*.](../14XSun_Analysis_of_incident_solar_radiation/14_01_Analysis_of_incident_solar_radiation_with_XSun.md)
<br>
<img src="./assets/20SimView - Værktøjsbjælken.png" width=15> Switch to [*tsbi5*.](../13tsbi5_thermal_simulation/13_01_tsbi5.md)
<br>
<img src="./assets/21SimView - Værktøjsbjælken.png" width=15> Start [*tsbi5Batch*](../13tsbi5_thermal_simulation/13_05_tsbi5_Batch.md)
<br>
<img src="./assets/22SimView - Værktøjsbjælken.png" width=15> Start plug-in application 1, in this case [*SimDXF*](../08SimDXF_CAD_drawings_as_basis_for_geometry/08_01_CAD_drawings_as_a_basis_for_geometry.md) (install via *<u>A</u>pplication* | *Setup*).
<br>
<img src="./assets/23SimView - Værktøjsbjælken.png" width=15> Start plug-in application 2, in this case *Notepad* (install via *<u>A</u>pplication* | *Setup*).
<br>
<img src="./assets/24SimView - Værktøjsbjælken.png" width=15> Start [*SimDB*.](../07SimDB_Database/07_01_The_SimDB_database.md)
<br>
<img src="./assets/25SimView - Værktøjsbjælken.png" width=15> Update view.
<br>
<img src="./assets/26SimView - Værktøjsbjælken.png" width=15> Zoom in (shortcut: "+").
<br>
<img src="./assets/27SimView - Værktøjsbjælken.png" width=15> Zoom out (shortcut: "-").
<br>
<img src="./assets/28SimView - Værktøjsbjælken.png" width=15> Expand bounding box by 20 % in all directions to show a larger area around the model.
<br>
<img src="./assets/29SimView - Værktøjsbjælken.png" width=15> Collapse bounding box by 20 % in all directions to show a smaller area around the model.
<br>
<img src="./assets/30SimView - Værktøjsbjælken.png" width=15> Move viewpoint (rotate the model) to the left (shortcut: "left arrow").
<br>
<img src="./assets/31SimView - Værktøjsbjælken.png" width=15> Move viewpoint down (view the model more from above) (shortcut: "up arrow").
<br>
<img src="./assets/32SimView - Værktøjsbjælken.png" width=15> Move viewpoint up (view the model more from below) (shortcut: "down arrow").
<br>
<img src="./assets/33SimView - Værktøjsbjælken.png" width=15> Move viewpoint (rotate the model) to the right (shortcut "right arrow").
<br>
<img src="./assets/34SimView - Værktøjsbjælken.png" width=15> Shows the *About BSim* dialog.
<br>
<img src="./assets/35SimView - Værktøjsbjælken.png" width=15> Select the [*View/ModelList*](../06BSim_Program_structure/06_07_SimView_Printing_a_model.md) menu option or click the corresponding icon. This will open a window in which the model's data is displayed line by line. Each line starts with an icon indicating the type of information. As long as this window is displayed, the model can be printed.
