# Help

</figure>

*   *Help Topics*: Opens the interactive help with the contents of the [default help file](https://bsim.outseta.com/support/kb/articles/LmJv8EmP/options) (English or Danish).

*   *Users Guide*: Opens user's guide in the form of a compiled HTML help file in English or Danish (if installed).

*   *BSim*: Provides access to a submenu where:

    *   *SBi Home Page* ... establish an Internet connection to the home page of Danish Building Research Institute.

    *   *BSim* establish an Internet connection to the home page of *BSim*.

    *   *Mail to BSim* ... opens an ordinary e-mail dialog to *BSim* where it is possible to ask [questions](https://help.bsim.dk/support/kb/articles/DmwABA94/sporgsmal-via-e-mail) related to *BSim*. Questions submitted from here will be distributed to all subscribers of the distribution list. **Note**: **Only** subscribers (to the English and/or Danish list) are allowed to send messages to the list. You can join the list from the BSim home page or from the next entry.

    *   *Subscribe BSim* ... opens an ordinary e-mail dialog with the necessary text to subscribe the senders e-mail address to the (English and/or Danish) BSim distribution list.

    *   *Check for new update* ... establishes an Internet connection and checks if a more recent version than the one currently running exists on the Internet. If a more recent version exists the version number of this update is shown in an message box. You must connect manually to the [BSim web-site](https://www.build.aau.dk/til-byggebranchen/software/bsim) to carry out the update.

*   *About BSim*: Opens a [dialog](https://bsim.outseta.com/support/kb/articles/VmAOk89a/about-bsim) box with information on *BSim* (version number, etc.), the associated modules (DLL) and a brief summary of the system information for the PC.
