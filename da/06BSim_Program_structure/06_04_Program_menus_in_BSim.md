# Programmenuer i BSim

Som i andre Windows programmer findes der øverst i programvinduet en række knapper, som kalder forskellige menuer frem ved venstre-klik med musen eller ved tryk på Alt-knappen efterfulgt af det bogstav, som er understreget i navnet for den ønskede menu, fx Alt-f for at åbne menuen <u>F</u>ile.
