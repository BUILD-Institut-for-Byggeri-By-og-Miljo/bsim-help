# Application

<figure id="center_img">
<img src="./assets/menu_application.jpg" alt="Menuen Application (Alt-a).">
<figcaption>Menuen Application (Alt-a).</figcaption>
</figure>

*   *Application navn(e)*: En liste med programmer som kan startes fra *BSim* med modellen som inddata.

*   *Setup*: Åbner en [dialog](../24Miscellaneous/24_19_Setting_up_applications.md) for valg af programmer, som ønskes at kunne starte direkte fra *BSim*. Programmerne optræder dels i menuen *Application* og dels som trykknapper på [værktøjsbjælken](../06BSim_Program_structure/06_05_SimView_Toolbar.md) i gruppen umiddelbart til højre for *tsbi5*-ikonen.
